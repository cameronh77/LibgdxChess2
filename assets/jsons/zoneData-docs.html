zoneData.json

Reference guide for authoring zones, encounters, and shops.

Top-level structure

The file is a JSON array. Each element is one zone — a named map with its own size, layout variants, enemy pool, and shop pool.

Variants

A variant is one hand-authored layout for the map. On each new run the game picks a random variant — so multiple variants give the player different map arrangements across playthroughs.

MapNode fields
FieldTypeDescription

refstringrequiredWhich encounter/shop to use. Either a specific ID or "random" — see below.

How ref works

Specific ID — e.g. "ref": "tollgate-4x4" — always loads that exact enemy or shop entry regardless of its fixed value.
"random" — picks a random entry from the zone's pool, but only from entries where fixed: false. Fixed entries are excluded.

Toll Gate — aura behaviour: Enemy pieces (the player's pieces) cannot cross the toll gate's row. A piece starting below the gate can move up to but not past its row. A piece starting on the exact same row is unaffected. The restriction updates live as the toll gate moves or is captured.

fixed: true vs false — at a glance

fixed: truefixed: false
Reachable via explicit ref ✅ Yes ✅ Yes
Included in "random" pool ❌ No ✅ Yes
Typical use Boss fights, unique encounters, scripted story nodes Filler / procedural encounters
Note: fixed has no effect on map layout. Node positions (x, y) are always fixed per variant — only the content of a node can be random.

Test encounters added new

Three toll gate encounters were added, one per zone, each referenced by a dedicated variant. They are all fixed: true so they never appear via "random" — only via the new variant.

tollgate-4x4 — classic4 (4×4 board)
{
  "enemyId": "tollgate-4x4",
  "fixed": true,
  "enemyLayout": [
    { "piece": "toll-gate",   "x": 2, "y": 2 },
    { "piece": "goblin-pawn", "x": 0, "y": 3 }
  ]
}

Toll gate at y=2. Goblin pawn at y=3 is unreachable until the gate is captured. Simplest case — good for first verification.

tollgate-6x6 — classic6 (6×6 board)
{
  "enemyId": "tollgate-6x6",
  "fixed": true,
  "enemyLayout": [
    { "piece": "toll-gate",   "x": 3, "y": 3 },
    { "piece": "goblin-pawn", "x": 1, "y": 5 },
    { "piece": "goblin-pawn", "x": 4, "y": 5 }
  ]
}

Gate at y=3 locks off the entire top half. Both pawns at y=5 are protected. Tests the aura covering multiple pieces simultaneously.

tollgate-8x8 — classic8 (8×8 board)
{
  "enemyId": "tollgate-8x8",
  "fixed": true,
  "enemyLayout": [
    { "piece": "toll-gate",    "x": 4, "y": 4 },
    { "piece": "goblin-pawn",  "x": 1, "y": 6 },
    { "piece": "goblin-pawn",  "x": 6, "y": 6 },
    { "piece": "goblin-queen", "x": 4, "y": 7 }
  ]
}

Full board. Gate at y=4 splits the board. Goblin queen at y=7 is maximally protected. Also exercises the interaction between the toll gate aura and the goblin queen's petrifying aura.
